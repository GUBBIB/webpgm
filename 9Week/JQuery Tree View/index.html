  • 문법기초
    • 1.1 변수
      • 1.1.1 변수 만들기
        • 1.1.1.1 여러분의 나이가 담긴 변수

          • <script>
                 var age = 30;
            </script>

        • 1.1.1.2 여러분의 이름, 나이가 담긴 변수

          • <script>
                 var name = "ddandongne";
                 var age = 30;
            </script>

        • 1.1.1.3 여러분의 이름, 나이가 담긴 변수 (콤마 활용)

          • <script>
                 var name = "ddandongne", age = 30;
            </script>

        • 1.1.1.4 변수 만들 때의 주의사항

          • <script>
            // 숫자로 시작 불가
                 var 1st = 10;

            // 대소문자 구분
                 var name = "ddandongne";
                 var Name = "differentName";

            // 예약어 사용 금지
                 var class = "test1";
            </script>
